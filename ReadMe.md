
# Google Drive Clone

A static clone of the Google Drive website


## Run Locally

Clone the project

```bash
  git clone https://github.com/Ameya29/Google-Drive-Clone
```

Go to the project directory

```bash
  cd Google-Drive-Clone
```

Go live from the HTML file
